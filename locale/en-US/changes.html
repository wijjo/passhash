Version 1.1.0

06-21-08

  • Made compatible with Firefox 3, thanks largely to code contributed by Kamil.
  • Support a greater range of hash word sizes.
  • Add a digits-only restriction for numeric hash words (e.g. for PINs).
  • Add preferences passhash.optShortcutKeyCode and passhash.optShortcutKeyMods as a backdoor to changing the assigned shortcut key and modifiers.
  • Add Italian translation contributed by Fabrizio.
  • Add German translation contributed by Robin Reisinger.
  • Handle .co.uk and similar country domains.
  • Focus on master key instead of site tag.
